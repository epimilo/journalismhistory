# Phong Trung Bay 360

Site static nay chay duoc tren GitHub Pages ma khong can build step.

## Cac file chinh

- `index.html`: khung trang, scene A-Frame va cac overlay.
- `styles.css`: giao dien HUD, bang thong tin va trang bao tuong tac.
- `script.js`: du lieu mau cho cac vat pham va logic mo dong.

## Day len GitHub Pages

1. Tao repo moi tren GitHub.
2. Day 3 file nay len nhanh `main`.
3. Vao `Settings` -> `Pages`.
4. Chon `Deploy from a branch`.
5. Chon nhanh `main` va thu muc `/root`.
6. Luu lai, doi GitHub Pages build xong la co link.

## Doi noi dung that

- Sua object `artifactContent` trong `script.js` de thay noi dung hotspot.
- Sua object `newspaperFeatures` trong `script.js` de thay bai viet tren trang bao.
- Neu muon dung anh that, dat file vao repo roi thay `src` hoac them asset trong `a-assets`.
